Flexbox Theory + Examples

Flexbox is a 1-dimensional layout system. It arranges items in a row or a column and makes alignment easy.

Core Theory

  • Flex Container: parent with display: flex;
  • Flex Items: direct children of that parent
  • Main Axis: direction of flex-direction (row by default)
  • Cross Axis: perpendicular to main axis
  • justify-content: aligns items on main axis
  • align-items: aligns items on cross axis

Most common confusion: axis changes when flex-direction changes. In row, main axis is horizontal. In column, main axis is vertical.

Detailed Options (What Each One Does)

1) flex-direction (defines main axis)

  • row (default): left to right
  • row-reverse: right to left
  • column: top to bottom
  • column-reverse: bottom to top

5) Item sizing (Beginner Friendly)

  • Step 1: flex-basis = starting width
  • Step 2: flex-grow = who gets extra free space
  • Step 3: flex-shrink = who is allowed to shrink when space is small
  • Shortcut: flex combines grow + shrink + basis

6) gap vs spacing values (common confusion)

  • gap adds fixed space between items only
  • space-between/around/evenly distribute remaining free space
  • Use gap for consistent spacing. Use justify-content for alignment and distribution.
